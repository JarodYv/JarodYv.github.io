高级

运用抽象和解构解决更复杂的问题

详细说明:

学员有能力解决更大的问题。通过运用抽象和解构以及更复杂的可复用数据结构，将大问题简化和分解。

学习成果

  • 能够将大问题分解为若干小问题并设计算法实现它们
  • 能够识别简单问题，并利用通用方法和抽象解决它
  • 能够有效地综合利用多个库或API的方法，并能查阅他们的文档
  • 能够写出可读性高的代码，在必要的地方加入注释

课程范例

  • Dress for the weather (Python)

  • Flappy astronaut (Python, Raspberry Pi, Sense HAT)

  • Getting started with the Twitter API (Python)
